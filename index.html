HTML

  • The head element contains information, or metadata about the webpage.
  • The body element represents the visible content shown to the user.
  • Child elements are nested inside parent elements.
  • Semantic elements describe their content and function to the developers and browser.
  • The difference between an element and a tag is, an element refers to the entire element from opening tag to closing tag, while a tag refers only to what appears inside the angle brackets.
  • Section elements define a section in a document that contains thematic content, such as this one.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • In a CSS rule, a selector defines the element or attributes to which the rules, or declarations, will apply.
  • Meanwhile, a declaration contains two important components: the property we want to apply, and the value of the property.
  • External style sheets are the preferred way to style a webpage due to the CSS file remaining separate from the HTML file, allowing for easier design changes.
  • The other two types of CSS style sheets are inline and internal.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Git is a version control system that lets you manage and keep track of each version of your project.
  • GitHub is a cloud-based hosting service that allows us to keep track of our projects inside repositories that are stored remotely.
  • The workflow for making content updates are as follows: create a branch, open a pull request, and merge and deploy.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Comments are snippets of text that can be added along with code, marked between /* and */.
  • Operators are mathematical symbols that produce results based on two values (or variables)
  • Conditionals are code structures used to test if an expression returns true or not. The most common form is the "if...else" statement.
  • Functions are a set of instructions that tell the computer how to perform a certain task.
